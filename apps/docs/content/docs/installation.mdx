## Getting Started with the template

Go to this repository's [github page](https://github.com/gabimoncha/rnr-registry-template) and click on the `Use this template` button. This will create a new repository with the template.

### Install the dependencies:

```bash
pnpm install
```

- Run the development server for both apps:

```bash
pnpm dev
```

- Run the development server for the docs app:

```bash
pnpm dev:docs
```

- Run the development server for the showcase app:

```bash
pnpm dev:showcase
```

- Build the registry:

```bash
pnpm registry
```
